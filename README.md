# Innsynsløsning for AAP

Ny innsynsløsning for AAP.

# Komme i gang

Appen kan kjøres lokalt med å kjøre følgende kommandoer

`yarn` - installere avhengigheter
`yarn dev` - starte appen

Appen kjører på port 3000 som default
